# Material de apoio (fontes autorais)

Coloque aqui o material de apoio para geração de questões.

- **Somente conteúdo autoral seu ou licenciado para uso comercial.** Nunca material
  oficial do CFA Institute, Learning Ecosystem, mocks oficiais ou questões de terceiros.
- Um arquivo por assunto. Markdown (`.md`) ou texto (`.txt`) de preferência.
- Sugestão de nome: `<TOPICO>-<assunto>.md` (ex.: `FSA-inventories.md`, `FI-duration.md`).
  Códigos de tópico: ETH, QM, ECON, FSA, CI, EQ, FI, DER, AI, PM.

> A ingestão destes arquivos (Agent 0 — Curriculum Mapper → objetivos internos →
> geração ancorada) é o próximo incremento sobre o pipeline da Fase 4.
